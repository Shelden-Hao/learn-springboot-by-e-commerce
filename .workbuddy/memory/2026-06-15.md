## Feature 9: 订单模块 — MyBatis XML 多表关联 + 动态 SQL
- schema.sql 新增 orders、order_items 表（一对多关系）
- 新建实体：Order、OrderItem、OrderStatus 枚举
- 新建 Mapper：OrderMapper（BaseMapper + XML）、OrderItemMapper（纯 BaseMapper）
- **核心文件：OrderMapper.xml** — 完整演示 MyBatis XML 语法：
  - `<resultMap>`：字段映射、`<id>`/`<result>`、`<collection>` 一对多嵌套
  - `<select>`：三表 LEFT JOIN（orders + order_items + products）
  - `<where>` + `<if>`：动态 SQL 条件拼接
  - `#{}` vs `${}` 安全对比
  - 速查手册：`<foreach>`、`<set>`、`<choose>`、`<sql>` + `<include>`
- OrderDetailDTO：嵌套 DTO（ItemDTO），作为 ResultMap 映射目标
- CreateOrderRequest：含嵌套 @Valid 校验、List<OrderItemRequest>
- OrderController：
  - POST /api/orders — 下单事务（校验库存→计算总价→生成订单号→插订单+明细→扣库存）
  - GET /api/orders — 动态条件分页（buyerName/status 可选）
  - GET /api/orders/{id} — XML 多表 JOIN 查订单详情
- DataInitializer 种子 3 笔示例订单
